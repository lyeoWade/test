Advantages

Reducing Delay and Broadening Bandwidth

Edge computing takes advantage of local deployment to process and store data in edge network. Decentralized layout requires less bandwidth and is closer to user terminals, so the delay is effectively reduced.

Location perception

When the terminal is connected to the wireless network, the local computing node can determine the geographical location of the device, identify the network needs of the user, and provide location-based and user-based analysis

localization

Edge computing platforms deployed locally are relatively independent, making it easier to leverage local resources and grow local services and applications.

Support device heterogeneity

Edge computing platform provides a new entry point to support diverse heterogeneous software devices.

Improve resource utilization

Many intelligent terminals are idle in non-working state. Edge computing can be utilized in wireless network to realize physical resource sharing.
